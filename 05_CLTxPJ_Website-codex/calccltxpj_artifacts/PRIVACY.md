# Política de Privacidade — CalcCLTxPJ

Última atualização: 2025-12-20

## 1. Objetivo
O site https://calccltxpj.com.br fornece um simulador CLT x PJ e conteúdo educacional.
As estimativas são **aproximações** e não constituem aconselhamento financeiro.

## 2. Dados coletados
### 2.1 Dados fornecidos pelo usuário
Os valores inseridos no simulador (ex.: salário, taxas) são usados para gerar o resultado na tela.

### 2.2 Analytics (opcional)
Podemos coletar métricas agregadas (ex.: cliques e uso de funcionalidades) para melhorar o produto.

## 3. IA (análise qualitativa)
Quando você solicita uma análise qualitativa por IA, os dados necessários para gerar a resposta podem ser enviados a um serviço de IA.
- Chaves de API não ficam no navegador.
- Podemos aplicar redaction/anonimização quando aplicável.

## 4. Compartilhamento
Não vendemos dados. Compartilhamentos ocorrem apenas para operar o serviço (analytics/IA), quando habilitados.

## 5. Contato
Defina aqui um e-mail de contato (ex.: suporte@calccltxpj.com.br).
